# Jibit.Net
تجربه‌ای متفاوت از پایداری سرویسها و سهولت اتصال به بیش از 30 سرویس دهنده بانکی، پرداخت و دولتی، با نرخ موفقیت منحصر به فرد، پیچیدگی‌های صنعت بانکی را فراموش کرده و با اطمینان بر کسب و کار خود تمرکز کنید


## سرویس استعلام و احراز هویتی جیبیت


### بخش استتعلام بانکی

- دریافت اطلاعات کارت بانکی
- تبدیل شماره کارت به شماره شبا
- تبدیل شماره حساب به شماره شبا
- دریافت اطلاعات حساب از طریق شماره شبا
- همخوانی کدملی و شماره کارت
- همخوانی کدملی و شماره شبا
- همخوانی نام و نام خانوادگی با شماره شبا
- همخوانی نام و نام خانوادگی با شماره کارت
- همخوانی نام و نام خانوادگی با شماره حساب


### بخش استعلام اطلاعات هویتی :

- همخوانی شماره موبایل با کدملی
- استعلام کدپستی
- ثبت احوال کامل
- بررسی مطابقت نام و نام خانوادگی،کد ملی و تاریخ تولد


طریقه نصب پکیج در پروژه توسط Package-Manager  :

```
Install-Package Jibit.Net -Version 1.0.2
```

حال تنها کافیست که یک شی از کلاینت بسازید :
```
JibitClient client = new JibitClient("your api key","your secret key");
```
